You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • /multiply?num1=*int 1*&num2=*int 2*
  • /volume_cylinder?radius=*radius*&height=height -- calculates volume of a radius
  • /pythagorean?side1=*side 1*&side2=*side 2* -- calculates the third side of a right triangle
